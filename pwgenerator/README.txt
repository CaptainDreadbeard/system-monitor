# Python File Encryption Tool

A simple but powerful file encryption utility built in Python using the
cryptography library. Supports:

- Fernet key file encryption
- Password-based key generation (PBKDF2)
- Master password encryption/decryption (no key file needed)
- Strong password generator
- Menu-driven CLI interface

This project was built to learn real-world cryptographic engineering concepts.
